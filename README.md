# MERN15
MERN Stack for POOSD Group 15



How to get started developing:

1. Clone the repo with:
```git clone https://github.com/Juanchochos/MERN15.git```

2. Change working directory to frontend and run npm install:
```cd frontend && npm install``

3. Do the same for the backend:
```cd backend && npm install```




To run the server frontend and backend do the following:
- Backend:
    - cd into backend folder and use ```npm run dev``` to start the development build
    - use ```npm run start``` to use the production build
- Frontend: 
    - cd into frontend folder and use ```npm run dev``` to start dev build
    - use ```npm run build``` to build the dist folder. This builds all the html and css from react.
