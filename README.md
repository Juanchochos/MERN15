# MERN15
MERN Stack for POOSD Group 15



How to get started developing:

1. Clone the repo with:
```git clone https://github.com/Juanchochos/MERN15.git```

2. Change working directory to fronted and run npm install:
```cd frontend 

    npm install``

3. Do the same for the backend:
```cd backend 
    npm install```
